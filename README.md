# earthquake-tsunami-analysis

## Project Title
Exploratory Analysis of Global Earthquake–Tsunami Events (2001–2022)

## Project Description
This project performs an Exploratory Data Analysis (EDA) on a global earthquake dataset to understand patterns and characteristics of earthquakes and tsunami-generating events between 2001 and 2022.

The goal of the analysis is to explore relationships between seismic features such as magnitude, depth, and geographic location, and to identify factors that may influence tsunami generation.

The project uses Python and visualization libraries to analyze the dataset and present insights through charts and statistical summaries.

---

## Objectives
- Analyze earthquake occurrences over time (2001–2022)
- Study the distribution of earthquake magnitude and depth
- Visualize geographic distribution using latitude and longitude
- Compare tsunami and non-tsunami earthquakes
- Identify patterns and insights related to tsunami generation

---

---

## Dataset Features
The dataset includes the following important features:

- Magnitude
- Depth
- Latitude
- Longitude
- CDI (Community Determined Intensity)
- MMI (Modified Mercalli Intensity)
- Significance Score
- Number of Stations
- Distance to Nearest Station
- Azimuthal Gap
- Year
- Month
- Tsunami (0 = No, 1 = Yes)

---

## Visualizations Performed
The following visualizations were created in the project:

- Year-wise earthquake frequency
- Tsunami events by year
- Magnitude distribution histogram
- Depth distribution histogram
- Magnitude comparison using box plots
- Geographic distribution using scatter plots
- Correlation heatmap of seismic features

---

## Key Insights
- Earthquake occurrences vary across different years.
- Tsunami-generating earthquakes are relatively rare compared to total earthquakes.
- Higher magnitude earthquakes have a greater probability of generating tsunamis.
- Earthquake locations show clusters in tectonically active regions.
- Magnitude and depth influence tsunami occurrence.

---

## Conclusion
The exploratory analysis highlights patterns in seismic activity and tsunami generation. Earthquakes with higher magnitudes and certain seismic characteristics are more likely to generate tsunamis. Visualization techniques help reveal relationships between seismic variables and improve understanding of earthquake behavior.
